Ayuda para el Plugin: Buscador de Caracteres Ocultos

¿Qué hace este plugin?

Este plugin permite buscar en la base de datos de WordPress fotos que contienen el carácter oculto '\u200E' en sus nombres. Utiliza un shortcode [jlmr_buscador_caracteres_ocultos] para activar la funcionalidad en cualquier página o entrada del sitio. Además, permite duplicar las fotos detectadas eliminando el carácter oculto del nombre de archivo.

¿Cómo funciona?

Al insertar el shortcode en una página o entrada, se muestran dos botones: uno para buscar caracteres ocultos y otro para duplicar las fotos afectadas. Al hacer clic en el primer botón, el plugin realiza una búsqueda en la base de datos de WordPress, específicamente en las fotos subidas (formatos JPEG y PNG), buscando el carácter oculto. Los resultados se almacenan en un archivo de texto.

El segundo botón permite duplicar las fotos identificadas, eliminando el carácter oculto de sus nombres, y las guarda en una carpeta específica dentro del directorio de subidas de WordPress.

Utilidad del Plugin

Este plugin es especialmente útil para identificar y corregir problemas relacionados con nombres de archivos de imágenes que contienen caracteres ocultos. Esto puede ser crucial para el SEO y la gestión de archivos.

¿Cómo se usa?

Para utilizar este plugin, sigue estos pasos:

  1. Inserta el shortcode [jlmr_buscador_caracteres_ocultos] en la página o entrada donde quieras utilizar la funcionalidad.
  2. Visita la página o entrada en tu sitio web y verás dos botones: "Buscar Caracteres Ocultos" y "Duplicar foto sin carácter oculto".
  3. Para iniciar la búsqueda de caracteres ocultos, haz clic en "Buscar Caracteres Ocultos".
  4. Para duplicar y limpiar las fotos detectadas, haz clic en "Duplicar foto sin carácter oculto".

Nota: Este plugin requiere que tu sitio web esté corriendo en WordPress y que tengas permisos para agregar shortcodes y modificar archivos.
